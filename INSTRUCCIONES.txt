DISTRIBUIDORA SOLMUNOL - CONFIGURACIÓN LOCAL CON MARIADB / MYSQL

Este proyecto fue reestructurado para usar MariaDB/MySQL en lugar de PostgreSQL.

1. Abre XAMPP.
2. Enciende Apache y MySQL.
3. Entra a http://localhost/phpmyadmin.
4. Crea una base de datos llamada:
   olmunol

   Collation recomendada:
   utf8mb4_unicode_ci

   También puedes ejecutar:
   database/sql/crear_bd_olmunol_mariadb.sql

5. Abre CMD o PowerShell en la raíz del proyecto, donde está artisan:
   C:\Users\HP\Desktop\erp_solis>

6. Ejecuta:
   composer install
   php artisan optimize:clear
   php artisan config:clear
   php artisan cache:clear
   php artisan route:clear
   php artisan view:clear
   php artisan migrate:fresh --seed
   php artisan serve

7. Entra a:
   http://127.0.0.1:8000/admin

CONFIGURACIÓN ACTUAL DEL .ENV

DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=olmunol
DB_USERNAME=root
DB_PASSWORD=

NOTA
Aunque uses MariaDB, en Laravel se usa normalmente DB_CONNECTION=mysql porque MariaDB trabaja con el driver pdo_mysql.
